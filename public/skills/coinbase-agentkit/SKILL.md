# Coinbase AgentKit

> "Every AI Agent deserves a wallet."

AgentKit is Coinbase Developer Platform's toolkit for giving AI agents a crypto wallet and onchain interactions. Framework-agnostic and wallet-agnostic with fee-free stablecoin payments.

## Install

```bash
# Node.js
npm install @coinbase/agentkit

# Python
pip install coinbase-agentkit

# Quick CLI setup
npx awal@latest                    # Init agentic wallet
npx skills coinbase/agentWallet    # Or install as agent skill
```

## Key Features
- **Framework-agnostic** — works with any AI framework (LangChain, OpenAI, Vercel AI SDK, CrewAI, MCP)
- **Wallet-agnostic** — Coinbase Smart Wallet, Embedded Wallet, External Wallet
- **Fee-free stablecoin payments** — monetize agents seamlessly
- Multi-chain: EVM + Solana support
- Python and Node.js SDKs

## Supported Protocols
Uniswap, Morpho, Aerodrome, Pyth, WOW, Jupiter, and more.
