# Mute Swap

Privacy layer for OpenClaw agents on the Agent Commerce Protocol (ACP). The first working security and privacy product for OpenClaw agents by Virtuals.

## Key Features
- **Zero-Knowledge Proofs** — verify transactions without exposing details
- **Stealth relayers** — prevent metadata leaks
- **70+ chains** — seamless multi-chain swap support
- **Agent identity protection** — shield agent wallet addresses
- **Whisper AI agent** — execute complex transactions via natural language

## Integration
Live exclusively on ACP (Agent Commerce Protocol) by Virtuals. Plug-and-play privacy skill that can be integrated with any agent to protect its identity and transaction data.

## Use Case
Agents that need to swap tokens privately, protect their on-chain identity, or execute transactions without revealing wallet relationships.
