# Turnkey Wallet

Wallets-as-a-Service with flexible policy engine. Agents get delegated signing within strict limits.

## Key Features
- Secure enclaves — keys never leave TEE
- Policy engine — spending limits, contract whitelists
- Delegated access — agents sign within boundaries
- Multi-approver quorums
- TypeScript SDK

## Install
```bash
npm install @turnkey/sdk-server
```
