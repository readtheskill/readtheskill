# Binance AI Agent Skills Hub

7 modular skills from Binance. Analyze markets, assess risks, and execute trades in one connected workflow.

## The 7 Skills

### 1. Binance Spot Skill
Real-time market data (exchange info, ticker prices, depth, candlesticks) + trade execution (order placement, cancellation, OCO/OPO/OTOCO orders). API key auth for mainnet and testnet.

### 2. Query Address Info
Analyze any wallet address: holdings breakdown, valuations, 24h changes, concentration. Smart money monitoring and whale tracking.

### 3. Query Token Info
Instant token metadata: symbol, blockchain, price, liquidity, holders, trading activity.

### 4. Crypto Market Rank
Aggregated rankings: trends, hot searches, smart money inflows, meme narratives, trader PnL. Structured "what to watch today" output.

### 5. Meme Rush
Track meme tokens across lifecycle phases (newly launched, migrating, migrated). Maps narrative themes to structured token lists.

### 6. Trading Signal
Smart money buy/sell signals with trigger price, current price, max gain, exit rate, signal status.

### 7. Query Token Audit
Auto-detect contract risk factors: minting capability, freeze functions, owner privileges.
