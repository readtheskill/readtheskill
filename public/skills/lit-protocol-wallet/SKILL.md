# Lit Protocol Wallet

Decentralized key management using PKPs — ECDSA keypairs as ERC-721 NFTs, secured by threshold MPC and TEEs.

## Key Features
- Programmable Key Pairs (PKPs)
- Threshold MPC + TEE security
- GOAT SDK plugin — `goat-sdk/wallet-lit`
- Social login support
- Multi-chain signing

## Install
```bash
npm install @goat-sdk/wallet-lit
```
